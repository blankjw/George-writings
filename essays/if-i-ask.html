IF I ASK

By George Gondron

IF I ASK If I ask, would you heal my anger that it might reveal my brokenness? If I ask, Would you heal my neediness that it might reveal my wealth? If I ask, would you heal my sadness that it might reveal my joy? If I ask, would you heal my desires that it might reveal my attachments? If I ask, would you heal my apathy that it might reveal my compassion? If I ask, would you heal my fear that it might reveal my darkness? If I ask, would you heal my judgment that it might reveal my own forgiveness? If I ask, would you heal my doubt that it might reveal my own faith? If I but ask.

George G

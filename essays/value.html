ESSAY

VALUE

By George Gondron

VALUE What is of value to some, is worthless to others What we value in our youth Becomes useless in our later years The foolish chase gratification and call it happiness The wise seek clarity and call it awareness What value is the past but to keep us in bondage What value is the future but to shackle us in fear What value is time if not lived in the now The golden gift of each present moment Most live as if time will never run out, but it will In the blink of an eye a life has run its course It runs very swiftly, does time It waits for no one What makes it precious, cherished, and prized Is people, relationships, connections Seeking solace from things alone blurs time Time becomes imperceptible as it wastes away It is only relationship that focuses the now That relates the value of each fragile moment There is no value without relationship All else will leave us empty and alone There is no meaning without human connection The rest is mere pretend fulfillment Perhaps if we knew ourselves more intimately Understood ourselves at the deepest level Especially what we chose not to see We would find what is of true value The value of ourselves in relation to others The value of ourselves in the mystery of Life The value of ourselves in communion with Love Perhaps we would, perhaps

George G
